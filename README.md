# 投诉链接快速打开器

## 用途
- 京东客服部内部专用工具
## 使用步骤
1. 打开 `http://jianguan.woaiwusaqi.cn`。
2. 选择 Excel 文件（支持 `.xlsx/.xls/.csv`）。
3. 点击“解析并生成链接”（解析后会自动保存到本地）。
4. 点击对应数据卡片可直接打开链接，也可使用 `Alt + 快捷键` 快速打开。
5. 点击卡片右上角删除图标可立即删除该条数据（无二次确认，删除后自动重排快捷键）。
6. 在“字段保留设置（DIY）”中勾选要保留显示的字段，未勾选字段会从前端列表隐藏。
7. 页面下次打开会自动从本地恢复上次数据。

## 字段 DIY 说明
- 支持客户根据导入数据自定义保留字段。
- 可点击“保留全部字段”或“仅保留核心字段（投诉单号、处理情况）”。
- 字段切换后表格会立即刷新，不影响链接生成逻辑。

## 上传合并与唯一约束
- 投诉单号是唯一主键：重复投诉单号不会被重复添加。
- 可重复上传多个文件：新文件数据会在旧数据基础上追加合并，不会覆盖或清空旧数据。
- 删除数据只能通过前端卡片右上角删除图标手动执行。



## 兼容说明
- 完全前端运行，不依赖后端。
- 依赖 CDN 加载 `xlsx` 解析库，首次使用需要联网。
- 本地持久化优先使用 IndexedDB，若不可用会降级到 localStorage。
- 点击“清空结果”会同时清空页面数据和本地缓存。
